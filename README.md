# ☁️My Cloud Computing Learning Journey

## Overview
This repository documents my 1-month intensive hands-on cloud computing practice using AWS Free Tier and Google Cloud Skills Boost.

The goal of this journey is to build strong cloud fundamentals, security awareness, and practical experience through daily labs and projects.

## Platforms Used
- AWS Free Tier
- Google Cloud Skills Boost
- GitHub

## Learning Structure
- Concept learning
- Hands-on labs
- Project building
- Documentation and reflection

## 1-Month Focus Areas
- Cloud fundamentals
- Identity and Access Management (IAM)
- Storage (S3)
- Compute (EC2)
- Networking & Security
- Monitoring & Logging
- Automation & Infrastructure as Code

## Outcome
By the end of this journey, this repository will serve as a public cloud portfolio demonstrating my foundational cloud and security skills.
